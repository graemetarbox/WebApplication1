Course Overview

Higher Diploma in Science in Cloud Computing

The course enables the learner with the necessary skills to develop a software system that is meant to be deployed as a Software as a Service to a cloud computing platform. The topics covered in Semester 1 include;

  • Object Orientated Software Engineering
  • Software Development
  • Web Design
  • Computer Architecture Operating Systems and Networks
  • Introduction to Databases
  • Career Bridge
